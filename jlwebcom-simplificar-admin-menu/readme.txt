== Description ==
Simplifica el escritorio de administración de WordPress a todos aquellos usuarios que no sean administradores.

Con ello les harás más sencilla su tarea, ya que únicamente verán aquellos apartados que necesitan utilizar.

Al mismo tiempo evitarás que cometan errores, modificando acidentalmente elementos que no les corresponde.

INSTALACIÓN:

Accede al escritorio de tu WordPress con tu usuario y contraseña.

En el menú de administración (vertical izquierdo) selecciona Plugins > Añadir Plugin > Seleccionar archivo > Instalar ahora.

Ahora, instalado y activa el plugin.

Por defecto de ocultarán algunos elementos del menú que, generalmente, no necesitarán utilizar.

Si lo deseas, puedes mostrar algunos de ellos u ocultar otros desde Ajustes > Simplificar menú. En la página de configuración podrás marcar o desmarcar las opciones que desees.

== Frequently Asked Questions ==
= ¿A qué usuarios afectará? =
Sólo a los que no tengan el nivel de privilegios de administrador.

= ¿Quién puede modificar la configuración del plugin? =
Sólo podran los administradores, eso sí, podrán hacerlo todos ellos.
